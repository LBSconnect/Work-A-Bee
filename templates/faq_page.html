Frequently Asked Questions (FAQ)

Welcome to the Work-A-Beez Frequently Asked Questions.

This page answers common questions about the platform today. For a broader look at where the product is headed, see the Product Overview and Feature Overview.

What is Work-A-Beez?

Work-A-Beez is a cloud-based workforce management platform that helps organizations manage employees, schedules, time and attendance, and leave from a single web application.

Is Work-A-Beez cloud-based?

Yes. Work-A-Beez is delivered as a Software-as-a-Service (SaaS) platform hosted in the cloud. No local software installation is required.

Do employees need to install anything?

No. Work-A-Beez is accessed through a supported web browser. See the Browser Support Guide and System Requirements for details.

Is there a mobile app?

Not yet. Work-A-Beez does not currently offer a dedicated native mobile application. The web application is responsive and works from mobile browsers.

Does Work-A-Beez support Multi-Factor Authentication (MFA) or Single Sign-On (SSO)?

Not yet. Both are planned enhancements to the authentication model. Today, accounts are secured with a username and password. See the Security Overview and Password Policy for current details.

Is an API available?

Not yet. A REST API is planned for a future release; it is not available to build against today.

How am I billed?

Billing depends on your selected subscription plan and billing cycle. See the Pricing Explanation, Billing FAQ, and Subscription FAQ for details.

Can I upgrade, downgrade, or cancel my subscription?

Yes. See the Subscription FAQ and Cancellation Policy for how plan changes and cancellations work.

Which browsers are supported?

Current versions of Google Chrome, Microsoft Edge, Mozilla Firefox, and Apple Safari. See the Browser Support Guide for full details.

I forgot my password. What do I do?

Self-service password reset is not yet available. Contact your organization's administrator, who can help you regain access, or reach out to the Work-A-Beez Support Center.

How do I contact support?

See the Support Center for how to reach the Work-A-Beez Customer Success team and what information to include with a request.

Where can I learn more about security and compliance?

See the Security Overview, Compliance Overview, and Data Protection Overview.
